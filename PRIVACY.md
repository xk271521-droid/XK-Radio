# Privacy

XK Radio 是本地桌面应用。项目本身不应把用户登录状态、Cookie、播放历史、搜索历史、自定义封面、自定义歌词、视觉用户存档或本地音乐上传到项目作者服务器。

## 本地保存的数据

- 网易云音乐、QQ 音乐等第三方平台登录状态和 Cookie。
- 搜索历史、播放历史、用户歌单缓存和视觉偏好。
- 自定义专辑封面、自定义歌词和用户导出的视觉存档。
- 本地节奏分析缓存，默认目录为 `D:\XKRadioCache\beatmaps`。

这些数据应只保存在用户本机，不应提交到 GitHub。

## 网络请求

XK Radio 可能会请求：

- GitHub Releases，用于检查和下载更新。
- Open-Meteo 等天气接口，用于天气电台体验。
- 网易云音乐、QQ 音乐等第三方平台接口，用于用户主动搜索、登录和播放体验。

## 第三方平台

用户通过第三方音乐平台登录时，应遵守对应平台的用户协议、版权规则和会员权益规则。XK Radio 不提供绕过付费、绕过会员、破解音质、批量下载版权音乐或重新分发音乐内容的能力。

## 开发注意

不要提交以下内容：

- `.cookie`、`.qq-cookie`、Token、账号信息。
- `node_modules/`、`dist/`、`updates/`。
- 用户上传音乐、缓存文件、导出的私人存档。
- 任何包含用户隐私或第三方平台凭据的日志。
